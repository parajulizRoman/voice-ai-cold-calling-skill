# Voice AI Cold Calling: Research-First Framework

A production-ready cold calling framework for voice AI agents (Vapi, Retell, ElevenLabs, etc.) selling Voice AI and automation services to local SMBs.

**Based on Katon Walker's proven method:** *"Do research, not sales."*

[Watch the source video](https://www.youtube.com/watch?v=CqE7lEaIckE)

## What's Inside

- **Full opening script** — "Bit of a weird call, I'm running a five-question survey..."
- **Five research questions** — Call volume, missed calls, ticket value, follow-up process, marketing channels
- **Pain calculation formula** — Turn missed calls into dollar amounts they can't ignore
- **Soft pitch technique** — "I have a friend in the UK..." (social proof without pressure)
- **Free demo offer** — The conversion mechanism
- **8 objection handlers** — Already have a service, retired, too small, AI skepticism, etc.
- **Voice AI agent implementation guide** — System prompt, tools, voice parameters, guardrails
- **Decision tree call flow** — Visual flow for the agent to follow

## Quick Start

Load `SKILL.md` as a knowledge base in your voice AI agent platform. See the "Implementation for Voice AI Agent" section for system prompt and tool setup.

## Results

In the source video: 25 businesses called, 20 reached, 1 demo built, multiple referrals — all in one hour, zero cost.
